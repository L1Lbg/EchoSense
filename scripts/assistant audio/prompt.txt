Respond with a JSON object structured as follows:

[
    {
        "action": "action_id",
        "parameters": {
            "param1": "value1",
            ...
        }
    },
    {
        "action": "action_id",
        "parameter":"content"
    }
]
Actions available:

write: Type with keyboard
presskey:  Press a keyboard key
openapp: Open an application
closeapp: Close an application
